Every WhatsApp Message — What the Member Sends & What They Receive Back
Member sends System action Reply sent to member
"Hi" / "Hello" / "Menu" / "0" / "Start" Creates/resets conversation state → MAIN_MENU "Welcome to eblockwatch. Please choose: 1. Community Update 2. Safety Tip of the Day ... 5. Cyber Chaperone Travel Safety ... 0. Main Menu"
"5" (from main menu) State → CYBER_CHAPERONE sub-menu "Cyber Chaperone — Travel Safety: 1. Start a new trip 2. Check active trip 3. Report I have arrived 4. Send SOS / need help 5. ETA update 6. Contact Andre 7. Back to main menu"
"1" (start new trip from CC menu) State → TRIP_FLOW / WAITING_FOR_START_LOCATION "Where are you leaving from? (Send your location pin or type your starting point)"
[Sends starting location] State → WAITING_FOR_DESTINATION "Got it. Where are you heading to?"
[Sends destination] State → WAITING_FOR_ETA "What time do you expect to arrive? (e.g. 18:30)"
[Sends ETA time] Creates GREEN trip · Route calculated · Checkpoints set · Operator mirror sent "✅ Trip registered. Andre is watching. Safe travels. Reply 'arrived' when you get there or 'help' if you need us."
"START Fourways to Rosebank ETA 19:40" Structured parser creates GREEN trip immediately "✅ Trip registered — Fourways → Rosebank, ETA 19:40. Andre is watching. Safe travels."
"Leaving [X] heading to [Y] ETA [Z]" Freeform parser creates GREEN trip "✅ Trip registered. Andre is watching. Safe travels."
ETA passes — no check-in (drift ≥15 min) System → CHECKIN flow · AMBER status · Operator mirror "⏰ You're 15 min past your ETA. Are you okay? 1. Yes, all good 2. Delayed 3. New ETA 4. I've stopped 5. I need help 6. Send my location pin"
"1" (from CHECKIN — I'm okay) Trip → GREEN · Last check-in time updated "✅ Thanks, noted. Andre can see you're okay. Stay safe."
"2" or "3" (delayed / new ETA) Trip → AMBER · New ETA collected "Understood. What is your new expected arrival time?"
Drift ≥45 min — no response Auto ICE escalation · Trip → AMBER · Message sent to ICE contact · Operator mirror (To ICE contact): "This is eblockwatch Cyber Chaperone. [Member name] set off at [time] and has not checked in. Please reply: 1. They're okay 2. They need help 3. I can't reach them 4. Please call me"
"Help" / "SOS" / "I need help" / "emergency" / "danger" Trip → RED · If ICE not yet contacted → auto ICE escalation · Operator mirror sent to Andre "🚨 SOS received. Andre has been alerted. ICE contact notified. Help is on the way. Stay on the line."
"Arrived" / "I have arrived" / "I'm home" Trip → COMPLETED · Trip closed · Operator mirror sent "✅ Safe arrival confirmed. Trip closed. Thank you for travelling with Cyber Chaperone."
Unknown member messages Message stored · Operator mirror to Andre · No auto-reply (No auto-reply — Andre sees it on the Situation Room dashboard)
