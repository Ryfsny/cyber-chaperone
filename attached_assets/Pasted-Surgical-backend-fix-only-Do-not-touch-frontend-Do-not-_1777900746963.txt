Current problem:

Required fix:
In the unknown branch, send a simple member acknowledgement.

Default unknown reply:
"Update received. Your trip is still being monitored. Please send ETA changes, delays, arrival, or help if needed."

If the message contains guidance/location wording such as:
- what should I do
- send location
- location
- pin
- where am I
- current location

Then reply:
"Yes. Please send your current WhatsApp location pin or Google Maps location link. We will add it to your active trip."

Keep the trip status unchanged unless the message contains Amber or Red keywords.

ISSUE 2 — Capture WhatsApp native location pins

Current problem:
A WhatsApp location pin arrived with Body = "".
The webhook did not read Latitude, Longitude, Address, or Label.
The operator mirror showed Message: "" and the location was lost.

Required fix:
Read location fields from Twilio req.body:
- Latitude
- Longitude
- Address
- Label

If Latitude and Longitude exist:
1. Treat the inbound message as a location update.
2. Link it to the latest active trip.
3. Update updatedAt / lastUpdateTime.
4. Add clean evidence note:
   [timestamp] LOCATION received: lat,lon [label/address if available]
5. Send member reply:
   "Location received. We have added it to your active trip."
6. Send Andre Operator Mirror:
   Title: CYBER CHAPERONE — LOCATION UPDATE
   Include:
   - member WhatsApp number
   - trip title
   - trip ID
   - latitude
   - longitude
   - address/label if available
   - Google Maps link created from the coordinates
   - status

Do not store empty evidence notes like:
Update received: ""

If Body is empty and no location fields exist:
- save as unknown empty message
- mirror to Andre as "Empty message received"
- do not crash

For plain Google Maps links sent as text:
- if Body contains maps.google.com, google.com/maps, or a URL with q=lat,long
- treat it as a location link update
- save clean evidence note:
  [timestamp] LOCATION LINK received: "<short URL/excerpt>"
- send member reply:
  "Location link received. We have added it to your active trip."
- mirror to Andre with the link visible.
